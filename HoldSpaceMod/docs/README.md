# HoldSpaceMod for Risk of Rain 2

## Description

HoldSpaceMod improves the auto-jump mechanic in Risk of Rain 2. Instead of instantly triggering auto-jump, the mod distinguishes between a short tap and a long press of the Space key, making gameplay smoother and more natural.

## Features

- Smart detection of short tap or long press on the Space key
- Auto-jump is only activated when holding the key
- Compatible with the latest versions of Risk of Rain 2

## Installation

1. Download the HoldSpaceMod.dll file from the Thunderstore page or the project release section.
2. Place the file in the `BepInEx/plugins` folder of your Risk of Rain 2 installation.
3. Launch the game as usual.

## Usage

- Tap Space for a normal jump.
- Hold Space to enable auto-jump as long as the key is held.

## Dependencies

No external dependencies required (except BepInEx and R2API, which are needed for most RoR2 mods).
