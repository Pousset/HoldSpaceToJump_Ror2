# HoldSpaceMod for Risk of Rain 2

## Description

HoldSpaceMod améliore la mécanique de saut automatique dans Risk of Rain 2. Au lieu de déclencher l'auto-jump instantanément, le mod fait la différence entre un appui court et un appui long sur la touche Espace, rendant le gameplay plus fluide et naturel.

## Fonctionnalités

- Détection intelligente de l'appui court ou maintenu sur la touche Espace
- Activation de l'auto-jump uniquement lors d'un appui prolongé
- Compatible avec les dernières versions de Risk of Rain 2

## Installation

1. Téléchargez le fichier HoldSpaceMod.dll depuis la section Thunderstore ou la release du projet.
2. Placez le fichier dans le dossier `BepInEx/plugins` de votre installation Risk of Rain 2.
3. Lancez le jeu normalement.

## Utilisation

- Appuyez brièvement sur Espace pour un saut normal.
- Maintenez Espace pour activer l'auto-jump tant que la touche est maintenue.

## Dépendances

Aucune dépendance externe requise (hors BepInEx et R2API, nécessaires pour la plupart des mods RoR2).
